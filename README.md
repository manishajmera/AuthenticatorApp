# Authenticator Full Stack APP

## Tech Stack used
1. Node(backend)
2.Mysql (database)
ReactJs(Frontend)
# Getting Started with Backend
1. Move to authenticator_backend foler and run `npm i` \n
2.`node index.js`

# Getting Started with Frontend
1. Move to authenticator-app foler and run `npm i` \n
2.`npm run start`
